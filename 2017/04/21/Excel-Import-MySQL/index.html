Hi，I‘m Zaki，最近正在学习python，进行inception自动化项目的开发，今天正在激情的开发中，突然一个开发找我，说明了需求，是需要将一个excel里面的数据导入到mysql表中，那么，今天就来看看，如何把excel里面的内容导入mysql表中。

  • excle转码成csv

打开excle，另存为文件，文件格式处选择CSV UTF-8的文件格式

  • csv文件生成服务器端的txt文件

1、使用command+c、command+v将csv文本文件复制出来

2、服务器端使用vim编辑一个文档，将内容复制进去

  • 新建对应列的表

查看excle文件有多少列，每列是需要什么类型，就去新建这么多列的一个表
